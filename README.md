# IT Asset Manager — Standalone (Packaged Executable via GitHub Actions)

This repo builds Windows/macOS/Linux executables with **no Node install required**.

## Quick start
1. Create a new GitHub repo and upload these files.
2. Go to **Actions** → `build-binaries` → Download the artifact for your OS.
3. Run the downloaded executable and open http://localhost:4000

## Local dev (optional)
```bash
npm install
npm start
```
